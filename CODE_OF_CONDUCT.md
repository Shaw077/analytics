# Code of Conduct

This project is committed to a harassment-free experience for everyone.

## Our Standards

- Be respectful and constructive.
- Assume positive intent.
- Welcome diverse perspectives.
- Focus feedback on ideas and code, not people.

## Unacceptable Behavior

- Harassment, insults, or discrimination.
- Personal attacks or trolling.
- Publishing private information without consent.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, and contributions that violate this code of conduct.
